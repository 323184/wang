### N1脚本

dc1.sh是安装dc1服务器的脚本

服务端安装
```
wget https://raw.githubusercontent.com/ZXQ-Kyle/N1Script/master/dc1.sh
bash dc1.sh
```

服务端卸载
```
wget https://raw.githubusercontent.com/ZXQ-Kyle/N1Script/master/dc1.sh
bash dc1.sh uninstall
```

手机使用8800端口连接服务器

全部文件都在files文件夹中
